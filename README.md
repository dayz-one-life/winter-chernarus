# Winter Chernarus
Winter textures and weather for Chernarus in DayZ.

## Installation Method 1: Local Script
Use this method if you have direct file access to your server (or have it mounted locally).
1. Download this repository to your computer.
2. Ensure you have Python 3 installed.
3. Run the installer from the terminal:
``` 
python install.py
```
4. Enter your server's mission directory path when prompted (e.g., C:\DayZServer\mpmissions\dayzOffline.sakhal).
5. Restart your server and enjoy!
___

## Installation Method 2: Web Installer (GitHub Actions)
Use this method to install directly from GitHub to your FTP server.

1. Fork this repository to your own GitHub account.
2. Go to the Actions tab.
3. Select "Install to Server" from the left sidebar.
4. Click "Run workflow".
5. Enter your FTP Credentials and your Mission Path.
> Note: The Mission Path must be the absolute path to the mission folder on your FTP server (e.g., /dayzxb_missions/dayzOffline.sakhal).
6. Click Run workflow.
7. Restart your server and enjoy!
